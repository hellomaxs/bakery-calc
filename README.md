# Пекарня · Калькулятор стоимости

PWA-калькулятор продажной стоимости изделий пекарни. Работает офлайн, устанавливается на домашний экран iPhone и Android.

## Вкладки

- **Витрина** — карточки изделий (фото, продажная цена, выход), фильтр «Выпечка / Напитки». Тап по карточке — состав изделия.
- **Тех карты** — база изделий: технологическая карта (брутто / нетто / цена / сумма по каждому компоненту), себестоимость, наценка в %, галочка «Выставить на витрину».
- **Склад сырья** — компоненты с единицей измерения (г / мл / шт), количеством и ценой упаковки. Цена за единицу и все тех карты пересчитываются автоматически.

## Расчёты

- Себестоимость = Σ (брутто × цена за единицу сырья)
- Выход = Σ нетто (г / мл)
- Продажная цена = себестоимость × (1 + наценка%), округление вверх до рубля

## Технически

Vanilla HTML/CSS/JS, без сборки и зависимостей. Данные хранятся в localStorage устройства, есть экспорт/импорт JSON (вкладка Склад). Service worker кеширует приложение для офлайн-работы.

Локальный запуск: любой статический сервер, например `python -m http.server 8741` в каталоге проекта.
